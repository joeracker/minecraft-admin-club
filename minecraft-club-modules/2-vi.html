Intro to vi

"To me vi is Zen. To use vi is to practice zen. Every command is a koan. Profound to the user, unintelligible to the uninitiated. You discover truth everytime you use it." --Reddy

About Vi?

  • Created in 1978
  • By Bill Joy (father of Java)
  • Basic text editor that is everywhere!

Using vi

  • h,j,k,l - move left, down, up, right
  • :w - save file
  • :q - quit vi
  • i - enter editing mode (Insert)
  • esc - exit editing mode

vi keyboard

Linux

  • Written in 1991
  • By Linus Torvalds
  • Originally named "Freax"
  • The source code is free
  • Linux powers the cloud

Linux Demo

  • pwd - where am I?
  • ls - list directory contents
  • cd [directory to change to]- change directory
  • cp [copy from] [to]
  • vim [file to edit]
  • mkdir [name of new directory]
